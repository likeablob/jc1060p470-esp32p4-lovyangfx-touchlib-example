# JC1060P470 Dev Board Touchscreen Example

A simple touchscreen example for Guition JC1060P470 dev board using LovyanGFX and TouchLib.

## Hardware

- Board: JC1060P470 (ESP32-P4, 7" 1024x600 LCD, 32MB PSRAM, 16MB Flash)
- Display Driver: JD9165
- Touch Controller: GT911 (800x480)

## Setup

```bash
pio run
pio run -t upload
```
